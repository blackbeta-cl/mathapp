# MathApp

App web de estudio para ninos de 3 basico en Chile, enfocada en las tablas del 1 al 10 con juegos, desafios, historial y configuracion para priorizar tablas especificas.

## Incluye

- Modulo `Math > 3 basico > Multiplicaciones`
- Tres juegos iniciales: escribir, elegir y responder con voz
- Sesiones de 6 preguntas
- Cronometro y registro de tiempo por partida
- Historial local con puntajes, precision y clasificacion
- Configuracion persistente para priorizar tablas

## Desarrollo

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```

## Deploy

El repositorio incluye workflow de GitHub Pages en `.github/workflows/deploy-pages.yml`.
